
## 一、AI算力基础设施主线

### 1. AI芯片 / 国产算力 / GPU / DCU / ASIC

这是算力链的“脑”，但A股里真正纯正的国产AI训练芯片标的并不多，很多是生态、服务器、模组或解决方案公司。

|细分概念|逻辑|代表股票池|
|---|---|---|
|国产AI芯片 / GPU / DCU|国产替代、训练/推理芯片、昇腾/国产生态|寒武纪-U、海光信息、景嘉微、龙芯中科、复旦微电、国芯科技|
|AI加速卡 / 模组 / 板卡|芯片落地到服务器形态|工业富联、浪潮信息、中科曙光、神州数码、拓维信息|
|国产算力生态 / 昇腾生态|华为昇腾、国产大模型适配、服务器与行业解决方案|拓维信息、软通动力、润和软件、常山北明、神州数码、四川长虹|
|ASIC / 定制芯片 / IP|云厂商自研芯片趋势，A股多为IP、设计服务或接口芯片|芯原股份、华大九天、概伦电子、广立微、澜起科技|

**调研重点：**别只看“国产AI芯片”标签，要看芯片是否量产、是否进入真实集群、客户是谁、推理/训练订单是否能变成收入。

这是“卖算力”的资产层。2024年以来很多算力租赁个股经历过题材炒作，后续要重点区分**真实机柜资源、GPU资源、客户合同、回款质量**。
